# 🏠 House Price Prediction System (MLOps)

## 📌 Overview
This project is an **end-to-end House Price Prediction System** built using **Machine Learning and MLOps practices**. It predicts house prices based on various features and is deployed as a **production-ready API** with CI/CD automation.

---

## 🎯 Problem Statement
Accurate house price prediction helps:

- Real estate buyers & sellers  
- Investment decision-making  
- Market trend analysis  
- Property valuation  

This system predicts house prices using structured data and serves predictions via an API.

---

## 📊 Dataset
- Ames Housing Dataset (Kaggle)

### Features Used:
- Overall Quality  
- Living Area  
- Garage Capacity  
- Basement Area  
- Year Built  
- Year Remodeled  

---

## 🔁 End-to-End Pipeline

1. Data Ingestion  
2. Data Preprocessing (Pipeline + ColumnTransformer)  
3. Model Training (XGBoost)  
4. Model Evaluation (RMSE)  
5. Experiment Tracking (MLflow)  
6. Model Saving (.pkl artifacts)  
7. API Development (FastAPI)  
8. Dockerization  
9. CI/CD Automation (GitHub Actions)  
10. Deployment (Render)
